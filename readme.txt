== Description ==

With WP Mailgun SMTP plugin you can connect Mailgun to your WordPress website for sending emails. It bypasses the normal WP mail function and sends email using Mailgun service.

Simply Add Username and Password and start delivering emails through Mailgun.

You can also change the From Email address and From Name.

###Below are the list of settings that you have to add to start using the plugin:

2. Hostname: Hostname field will be automatically pre-filled.
3. Username: Add SMTP username that's been provided by Mailgun for sending emails.
4. Password: Add SMTP password to connect with Mailgun.
5. Protocol: Protocol field will be automatically pre-filled.

You can also try sending a test email with the plugin to check whether all your configurations are working correctly.


== Installation ==

1. Upload the plugin to your WordPress site.
2. Activate the plugin.
3. Set the options like From Email, From Name, Return Path, Username and Password etc, inside the plugin options by going to Settings -> Mailgun SMTP
4. Send a test email to check everything is working properly.
5. Enjoy
